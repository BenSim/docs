Introduction

In this tutorial you will learn how to build a CosmJS based dApp. The example dApp will be a balance checker that will allow you to see your native tokens and the CW20 tokens of the contract with the address you enter.

Views#

It will look like this:

Login#

Native balance#

Balance of a CW20 contract#

Error for address with no contract#

Setup environment#

We recommend to use Visual Studio Code but this tutorial should be easily followed with any other text editor.

You should download the latest release of the CosmWasm/dApps monorepo using your preferred method.
